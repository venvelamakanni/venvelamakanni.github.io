Customer Lifetime Value for Dropbox

Prediction project for understanding the behavior of customers.

Analyzed SaaS subscription data to predict user behavior. Through segmentation and churn analysis, I found 'Very High' revenue users averaged 17.27-month subscriptions versus 5.83 months for 'Low' segment users. My machine learning model achieved an R-squared of 0.283, with session duration, support efficiency, and revenue segmentation as key predictors. This analysis offers insights to improve retention and optimize support strategies.
